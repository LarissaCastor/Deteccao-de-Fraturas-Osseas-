# Deteccao-de-Fraturas-Osseas-
Objetivos do Projeto
* Utilizar o dataset aberto BoneFractureYOLO para detecção de fraturas.
* Aplicar YOLO para identificar regiões fraturadas nas imagens.
* Realizar pré-processamento, padronização e transformação linear das imagens (rotação, escala, espelhamento e translação).
* Executar PCA para:
* Reduzir dimensionalidade,
* Avaliar variância explicada,
* Reconstruir imagens com diferentes números de componentes,
* Visualizar agrupamentos (PC1 vs PC2),
* Investigar padrões latentes que discriminam fratura vs não-fratura.
